# Images for the main page

The main page (`index.html`) uses images in these places:

| Location | Purpose | Current source |
|----------|---------|----------------|
| **Hero (right column)** | Supportive couple / calm theme | Unsplash (external URL) |
| **What's inside** | Book / reading theme | Unsplash (external URL) |

To use your own images instead:

1. Add your image files here (e.g. `hero.jpg`, `section-book.jpg`).
2. In `index.html`, replace the `src` of each `<img>`:
   - Hero: look for `hero-image-wrap` and set `src="assets/hero.jpg"` (or your filename).
   - Section: look for `section-image-wrap` and set `src="assets/section-book.jpg"` (or your filename).

Use landscape or portrait images that fit the theme (supportive partner, reading, calm). Recommended size: at least 760px wide for the hero, 1120px wide for the section image.
